    virtuoso操作—以反相器为例

    DNFfafa

    # virtuoso操作—以反相器为例

    # 打开virtuoso

    1、右键新建终端

    2、在终端运行mkdir project,新建文件夹project

    3、在终端运行cd project和mkdir inv，到project文件夹下，新建inv文件夹

    4、在终端运行cd inv，进入inv文件夹

    5、在终端运行virtuoso &打开virtuoso界面。&的意思是：在后台运行

    # 新建库

    1、在virtuoso界面，点击Tools→Library Manager打开Library Manager。

    2、Library Manage在里面点击 File—>New—>Library打开 New Library 对话框

    3、在对话框中输入新建库的名字，这里命名为 mycell，点击 OK。

    4、出现为新建库选择工艺库文件的对话框，点击第三项“Attach to ...... library”（关联工艺库），点击 OK，再选择对应的的工艺库（这里选择tsmc18rf）点击OK。

    5、最后在Library Manage里选择File->New->Cell View创建原理图，在Cell处命名为inv。

    # 原理图的绘制

    1、按快捷键i（小写）放置器件，选择Browse，在工艺库中选择所需要的元器件，例如：nmos2v、pmos2v，双击右侧的symbol放置。

    2、放置时，可以修改器件参数

    3、放置gnd和vdc：按快捷键i，选择Browse，在analogLib中，选gnd和vdc

    4、按快捷键p放置端口，分别放置vin、vout、vdd、gnd四个端口。其中vout我为output，其他为input

    5、搭建如图所示电路图，并保存

    6、为电路创建 symbol， 方便调用和搭建测试电路。 点击 Creat—>Cellview—>From Cellview

    7、点击OK，左边Pin为vin、右边为vout、上边为vdd、下边为gnd；点击OK，出现初始的symbol

    8、将symbol修改并保存

    9、在 Library Manager 可以看到 inv已生成一个 Symbol

    # 电路测试

    1、在Library Manager中， 选中 mycell， 另新建一个 Cell View ， 命名为inv_test。 2、快捷键i放置绘制好的Symbol 3、搭建如图所示的电流图。
